Cayman also offers powerful support for code snippets:

#### Ruby
{% highlight ruby %}
def print_hi(name)
  puts "Hi, #{name}"
end
print_hi('Tom')
#=> prints 'Hi, Tom' to STDOUT.
{% endhighlight %}

#### Python3
{% highlight python %}
def print_hi(name)
  print("Hi, " + name)

print_hi('Tom')
#=> prints 'Hi, Tom' to STDOUT.
{% endhighlight %}
